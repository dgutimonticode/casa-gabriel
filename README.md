# Casa Gabriel 🏡

Landing page inmobiliaria desplegada automáticamente en Amazon S3 con GitHub Actions.

## Tecnologías
- HTML5, CSS3, JavaScript vanilla
- Amazon S3 (Static Website Hosting)
- Amazon CloudFront (CDN)
- GitHub Actions (CI/CD)

## Despliegue
Cada push a `main` sincroniza automáticamente los archivos con el bucket S3.
